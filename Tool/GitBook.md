# GitBook

- 官方文档: http://toolchain.gitbook.com/

- 发现图书: https://www.gitbook.com/explore

- 安装
```
npm install gitbook-cli -g
```

- 初始化
```
gitbook init
gitbook init book_folder
git init
```

- 预览
```
gitbook serve
```

- 构建
```
gitbook build
gitbook build ./ --log=debug --debug
```

- 安装插件
```
gitbook install
```

- 常用插件
```
styles-sass
github
katex
```
